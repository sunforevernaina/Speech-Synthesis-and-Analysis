The transfer function in time domain for the single formant resonator is:

$H(S)=\frac{G}{(S+\sigma_{p}+\Omega_{p})(S+\sigma_{p}-\Omega_{p})}$

Poles are at $S=-\sigma_{p} \pm j\Omega_{p}$

Now we convert this transfer function to discrete domain by using Impulse Invarience Method.

$Z=e^{ST_{s}}$, where $T_{s}$ is the sampling time period.

$Z_{p}=re^{j\theta}$

$r=e^{-\sigma_{p}T_{s}}$ and $\theta= \Omega_{p}T_{s}$

$\Omega_{p}=2\pi F_{i}$ and $\sigma_{p}=\pi B_{i} $

$r_{i}=e^{-B_{i}\pi T}$ and $\theta_{i}=2\pi F_{i}T$

$H(Z)=\frac{k}{1-2r\cos \theta Z^{-1}+r^{2}Z^{-2}}$

Given in Question- $F1(formant frequency)=900Hz, B1(bandwidth)=200Hz$, $F_{s}=16KHz$

From this we calculate, $r=0.9615$ and $\theta=(9/80)\pi$

After putting values of $r$ and $\theta$, we get our Z domain transfer function as:

$H(Z)=\frac{1}{1-1.804Z^{-1}+0.924Z^{-2}}$

#=========importing required libraries================
import cmath as cm
import numpy as np
import matplotlib.pyplot as plt
# finction for plotting transfer function
def H(z):
    num = 1
    denom=1-1.804*z**(-1)+0.924*z**(-2)
    return num/denom
j=cm.sqrt(-1)
w_range = np.linspace(-np.pi, np.pi,8000)
magnitude= np.abs(H(np.exp(1j*w_range)))
log_magnitude=np.log(np.abs(H(np.exp(1j*w_range))))
fig, axes = plt.subplots(1, 2, figsize=(10,4))
      
axes[0].plot(w_range, magnitude)
axes[0].set_title("Normal scale")

axes[1].plot(w_range,log_magnitude)
axes[1].set_title("Logarithmic scale (y)")

plt.show()

$y(n)-1.804y(n-1)+0.924y(n-2)=x(n)$

now, when we put x(n)=$\delta(n)$, we get: $y(n)=h(n)$ ,where h(n) is impulse response.
